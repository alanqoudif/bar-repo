# 🌐 **المرشد الذكي للسياحة في سلطنة عُمان** ## 🌍 مقدمة "المرشد الذكي" هو منصة مبتكرة تهدف إلى تسهيل اكتشاف المعالم السياحية في سلطنة عُمان باستخدام تقنيات الذكاء الاصطناعي. توفر المنصة تجربة متكاملة للمستخدمين للاستفادة من توصيات مخصصة للأماكن السياحية، سواء عبر الإنترنت أو من خلال نسخة محلية تعتمد على نموذج ذكاء اصطناعي مدمج. --- ## ✨ **مميزات المنصة** 1. **🌿 توصيات مخصصة**: - المنصة تقدم اقتراحات للأماكن السياحية بناءً على اهتمامات المستخدم. 2. **🔧 دمج تقنيات الذكاء الاصطناعي**: - استخدام تقنيات OpenAI للحصول على إجابات دقيقة وفورية. - توفير نسخة محلية تعمل بدون اتصال إنترنت باستخدام نموذج "آية". 3. **🌍 واجهة مستخدم سهلة**: - تصميم جذاب وسهل الاستخدام يتناسب مع جميع الفئات العمرية. 4. **👤 دعم المرشدين السياحيين**: - إمكانية استخدام المرشدين السياحيين للمنصة لتنظيم الجولات وإدارة الحجوزات. 5. **📊 لوحة تحكم متكاملة**: - عرض بيانات المستخدمين، الأماكن المفضلة، ومراجعات الزوار بشكل تفصيلي. --- ## 🔍 **هدف المنصة** - دعم السياحة في سلطنة عُمان من خلال تسهيل عملية اكتشاف الأماكن السياحية. - تعزيز تجربة المستخدم باستخدام التكنولوجيا الحديثة. - مساعدة المرشدين السياحيين في تقديم خدمات عالية الجودة. --- ## 🚀 **التقنيات المستخدمة** 1. **WordPress**: - لإنشاء وإدارة واجهة المستخدم. 2. **OpenAI API**: - لتوفير ردود ذكية وسريعة بناءً على استفسارات المستخدمين عبر الإنترنت. 3. **نموذج محلي للذكاء الاصطناعي**: - يعمل محليًا بدون الحاجة لاتصال إنترنت. 4. **PHP وMySQL**: - لإدارة البيانات وتخصيص محتوى المنصة. 5. **HTML, CSS, JavaScript**: - لتطوير واجهة مستخدم تفاعلية وجذابة. --- ## 🤖 **كيف يعمل المرشد الذكي؟** 1. يقوم المستخدم بتسجيل الدخول إلى المنصة. 2. يحدد اهتماماته (مثل الطبيعة، التاريخ، المغامرات). 3. يقدم الذكاء الاصطناعي توصيات مخصصة بناءً على المدخلات. 4. يمكن للمستخدم الاستفادة من: - **ردود عبر الإنترنت** باستخدام OpenAI API. - **ردود محلية** عبر نموذج "آية" عند عدم وجود اتصال بالإنترنت. --- ## 🔹 **مزايا الذكاء الاصطناعي المستخدم** 1. **التواصل الفوري:** - الحصول على اقتراحات دقيقة وفورية. 2. **دعم بدون إنترنت:** - تشغيل المنصة محليًا باستخدام نموذج الذكاء الاصطناعي "آية". 3. **تكامل مع المرشدين السياحيين:** - تسهيل إدارة الجولات ومساعدة الزوار. --- ## 🏛️ **لماذا تختار "المرشد الذكي"؟** - **تجربة شخصية:** المنصة مصممة لتناسب احتياجات كل مستخدم. - **تقنيات متطورة:** دمج الذكاء الاصطناعي يجعل التجربة أكثر كفاءة. - **دعم السياحة المحلية:** المساهمة في تعزيز السياحة داخل سلطنة عُمان. --- ## 📡 **فريق العمل** - تم تطوير المنصة بواسطة فريق متخصص في التكنولوجيا والذكاء الاصطناعي لدعم قطاع السياحة في سلطنة عُمان. --- ## 🛠️ **كيفية تثبيت المنصة** 1. قم بتنزيل ملفات WordPress من هذا الريبو. 2. قم برفع الملفات إلى خادم الويب الخاص بك. 3. قم بإعداد قاعدة البيانات وتوصيلها بالمنصة. 4. قم بإضافة مفتاح OpenAI API إلى إعدادات المنصة لتفعيل الذكاء الاصطناعي عبر الإنترنت. 5. للتفعيل المحلي، قم بتثبيت نموذج "آية". --- ## 💡 **ملاحظات إضافية** -
